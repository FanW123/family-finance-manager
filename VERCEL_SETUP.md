# 📝 Vercel 注册和设置指南

## 是否需要注册？

**是的，需要注册 Vercel 账号才能部署。**

## 🎯 注册步骤

### 方法 1: 使用 GitHub 账号（推荐）

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击右上角 "Sign Up"

2. **使用 GitHub 登录**
   - 选择 "Continue with GitHub"
   - 授权 Vercel 访问你的 GitHub
   - 完成！

**优点：**
- ✅ 最快最简单
- ✅ 可以直接导入 GitHub 仓库
- ✅ 自动部署（代码推送后自动更新）

### 方法 2: 使用邮箱注册

1. **访问 Vercel**
   - 打开 https://vercel.com
   - 点击 "Sign Up"

2. **填写信息**
   - 输入邮箱
   - 设置密码
   - 验证邮箱
   - 完成！

## 🚀 注册后立即部署

注册完成后，你可以：

### 选项 A: 拖拽部署（最快）

1. 登录 Vercel Dashboard
2. 点击 "Add New" → "Project"
3. 直接拖拽 `client/dist` 文件夹
4. 完成！

### 选项 B: 通过 GitHub（推荐用于持续部署）

1. 推送代码到 GitHub
2. 在 Vercel 中导入仓库
3. 配置自动部署
4. 以后每次推送代码，自动更新！
